# Privacy Policy

**conjure** and all plugins in this repository are Claude Code extensions that run entirely on your local machine.

- No data is collected, transmitted, or stored by these plugins
- No analytics, telemetry, or usage tracking of any kind
- All operations occur locally within your Claude Code session
- No accounts, registration, or network requests are made by the plugins themselves

If you have questions, open an issue at https://github.com/john-farina/claude-plugins/issues
